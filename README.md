# Nocage Waitlist

**Access the Future.**

This is the exclusive waitlist landing page for **Nocage**, featuring a premium "Liquid Glass" design aesthetic.

## Features
- **Liquid Glass UI**: Ultra-realistic glassmorphism with dynamic lighting and blurs.
- **Seamless Animations**: Smooth entrance and interaction effects using Framer Motion.
- **Robust Backend**: 
  - **Supabase**: Primary database for secure user storage.
  - **SheetDB**: Real-time sync to Google Sheets.
  - **Resend**: Automated, beautifully designed welcome emails.

## Tech Stack
- Next.js
- Tailwind CSS
- Framer Motion
- Lucide React
